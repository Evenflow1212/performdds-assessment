Day Sheet — three periods
Dentrix — Day Sheet (year-to-date + last full year + year before)

How to run this report in Dentrix

  1. In the Office Manager, click the chair icon at the bottom toolbar (Day Sheet shortcut) — or Reports → Management → Day Sheet
  2. Set Provider range to <ALL> and Billing Type range to <ALL>
  3. Set Date Type to PROCEDURE Date (NOT Entry Date — important)
  4. Choose Format: Chronological Day Sheet
  5. Under Print Totals, check Include MTD totals and Include Provider Totals (skip extended MTD)
  6. Run the Day Sheet THREE times — once per period below — and upload each one
  7. Click Batch (not Print). When the report appears in the Batch Processor, double-click to open it.
📸
The Day Sheet runs 200+ pages. You only need the LAST PAGE (the totals row). Easiest path: navigate to the last page, take a screenshot of just that page (Cmd+Shift+4 on Mac, Snipping Tool on Windows), and upload the screenshot. Slower alternative: print the entire report to PDF and upload the PDF. Both work — the screenshot just uploads faster.

Patient AR Aging
Dentrix — Aging Report (last-page screenshot or full PDF)

How to run this report in Dentrix

  1. In the Office Manager, click Reports → Ledger → Aging Report...
  2. Set: All Guarantors, All Providers, All Billing Types, Standard Aging only
  3. Set Min Balance to Print to 1.00
  4. Set Minimum Days Past Due to Over 0
  5. Click OK, then Batch, open from Batch Processor
  6. You only need the last page — the per-patient detail in front of it is irrelevant. Easiest: scroll to the last page, take a screenshot of the TOTALS row (Cmd+Shift+4 on Mac, Snipping Tool on Windows), upload it. Alternative: print the full report to PDF and upload the PDF — both work, the screenshot just uploads faster.
⚠️
Important: Set "Min Balance to Print" to 1.00 — this excludes credit balances and zero-balance accounts so you only see patients who actually owe money.

Dentrix — Insurance Claim Aging Report (last-page screenshot or full PDF)

How to run this report in Dentrix

  1. In the Office Manager, click Reports → Ledger → Insurance Aging Report...
  2. Select: Dental (not Medical), All Patients, All Providers, All Carriers, Over 0 Days
  3. Leave "Exclude Partially Paid Claims" unchecked
  4. Click OK, then Batch, open from Batch Processor
  5. You only need the last page — the TOTALS ALL CLAIMS row. Easiest: scroll to the last page, take a screenshot of just the TOTALS row (Cmd+Shift+4 on Mac, Snipping Tool on Windows), upload it. Alternative: print the full report to PDF and upload the PDF.

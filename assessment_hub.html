Patient AR Aging
Dentrix — Aging Report (manual entry)

How to run this report in Dentrix

  1. In the Office Manager, click Reports → Ledger → Aging Report...
  2. Set: All Guarantors, All Providers, All Billing Types, Standard Aging only
  3. Set Min Balance to Print to 1.00
  4. Set Minimum Days Past Due to Over 0
  5. Click OK, then Batch
  6. Open from Batch Processor — scroll to the very last page
  7. Screenshot the TOTALS summary box at the bottom, then enter the numbers below
⚠️
Important: Set "Min Balance to Print" to 1.00 — this excludes credit balances and zero-balance accounts so you only see patients who actually owe money.

Scroll to the last page. The TOTALS box at the bottom looks like this:

Insurance AR Aging
Dentrix — Insurance Claim Aging Report (manual entry)

How to run this report in Dentrix

  1. In the Office Manager, click Reports → Ledger → Insurance Aging Report...
  2. Select: Dental (not Medical), All Patients, All Providers, All Carriers, Over 0 Days
  3. Leave "Exclude Partially Paid Claims" unchecked
  4. Click OK, then Batch
  5. Open from Batch Processor — scroll to the very last page
  6. Enter the numbers from the TOTALS ALL CLAIMS row below

📸 What to screenshot & enter

The last page shows PRIMARY, SECONDARY, and TOTALS ALL CLAIMS rows. Use TOTALS ALL CLAIMS:
